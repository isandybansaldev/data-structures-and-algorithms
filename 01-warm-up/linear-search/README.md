# Linear Search in an Array

Given an array of integers and a target value, write a function that searches for the target element in the array. If the target is found, return its index; otherwise, return -1.

```js
arr = [5, 2, 1, 8, 7, 0];
target = 8;
output = 3;
```

```js
arr = [5, 2, 1, 8, 7, 0];
target = 9;
output = -1;
```
