# Find the Largest Number in an Array

## Problem

Given an integer array `nums`, return the largest number present in the array.

You may assume that the array contains at least one element.

---

### Example 1

**Input:**

```text
nums = [5, 2, 1, 8, 7, 0]
```

**Output:**

```text
8
```

**Explanation:**

The largest element in the array is `8`.

---

### Example 2

**Input:**

```text
nums = [-10, -5, -20, -3]
```

**Output:**

```text
-3
```

**Explanation:**

Among all the elements, `-3` has the greatest value.

---

### Example 3

**Input:**

```text
nums = [4]
```

**Output:**

```text
4
```

---

## Constraints

- `1 <= nums.length <= 10^5`
- `-10^9 <= nums[i] <= 10^9`

---
