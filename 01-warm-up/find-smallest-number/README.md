# Find the Smallest Number

## Problem Statement

Given an array of integers `nums`, return the smallest number present in the array.

You may assume that the array contains at least one element.

## Example 1

**Input**

```text
nums = [5, 2, 8, 1, 9]
```

**Output**

```text
1
```

**Explanation**

`1` is the smallest number in the array.

## Example 2

**Input**

```text
nums = [-3, -7, -1, -5]
```

**Output**

```text
-7
```

**Explanation**

`-7` is the smallest number in the array.

## Constraints

- `1 <= nums.length <= 10^5`
- `-10^9 <= nums[i] <= 10^9`

## Follow-up

Can you solve it in **O(n)** time and **O(1)** extra space?
